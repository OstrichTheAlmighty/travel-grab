# Sources

Raw evidence trail — every URL, report, and reference used in this project.
Analysis and conclusions derived from these sources live in `market-research.md`.

Format per entry:
```
[Source Name](URL) — Accessed: YYYY-MM-DD — Reliability: High/Medium/Low — Summary: one line.
```

Reliability guide:
- **High** — primary source, peer-reviewed, official report, or verified by 2+ independent sources
- **Medium** — single reputable outlet, industry publication, or analyst report (unverified)
- **Low** — blog, forum, unverified claim — use for leads only, not as evidence
